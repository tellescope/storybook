# Breadcrumbs

Breadcrumbs are navigation elements that help users understand their location within a site hierarchy and navigate back to higher levels.

## Interactive Example

<Canvas of={BreadcrumbsStories.Default} />

## Basic Usage

```javascript
import { Breadcrumbs } from '@/Molecules/Breadcrumbs';
import { Home, Settings } from '@mui/icons-material';

// Basic breadcrumbs with 3 levels
<Breadcrumbs levels={3} />

// With icons
<Breadcrumbs levels={3} icons={[HomeIcon, SettingsIcon]} />
```

## API Reference

<ArgTypes of={BreadcrumbsStories} />

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `levels` | `1 \| 2 \| 3` | `3` | Number of breadcrumb levels to display (maximum 3) |
| `icons` | `React.ReactNode[]` | `[]` | Array of icon components to display after breadcrumbs (maximum 2) |
