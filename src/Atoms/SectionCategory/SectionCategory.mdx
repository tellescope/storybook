# SectionCategory

A customizable accordion component with add button functionality. This component extends Material-UI's Accordion component to provide a clean and functional section header.

## Features

- **Expandable/Collapsible Content**: Toggle visibility of content with a click
- **Add Button**: Quick action button for adding new items
- **Customizable**: Easy to configure title and content
- **Variants**: Supports expanded and selected states

## Usage

### Basic Usage

```jsx
import CustomAccordion from './SectionCategory';

function App() {
  return (
    <CustomAccordion>
      <p>This is the content inside the accordion.</p>
    </CustomAccordion>
  );
}
```

### With Custom Title

```jsx
import CustomAccordion from './SectionCategory';

function App() {
  return (
    <CustomAccordion 
      title="Task Categories" 
    >
      <ul>
        <li>Task 1</li>
        <li>Task 2</li>
        <li>Task 3</li>
      </ul>
    </CustomAccordion>
  );
}
```

### Expanded Variant

```jsx
import CustomAccordion from './SectionCategory';

function App() {
  return (
    <CustomAccordion 
      title="Expanded Section" 
      expanded={true}
    >
      <p>This section starts in an expanded state</p>
    </CustomAccordion>
  );
}
```

### Selected Variant

```jsx
import CustomAccordion from './SectionCategory';

function App() {
  return (
    <CustomAccordion 
      title="Selected Section" 
      selected={true}
    >
      <p>This section has a selected/highlighted appearance with gray background</p>
    </CustomAccordion>
  );
}
```

## Variants

### Expanded Variant

The `expanded` prop controls whether the accordion is initially expanded or collapsed. When set to `true`, the accordion will be expanded when first rendered.

### Selected Variant

The `selected` prop controls whether the accordion appears in a selected/highlighted state. When set to `true`, the accordion will have a light gray background and no bottom border.
